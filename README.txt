
SETUP GUIDE

1. Upload files to GitHub repo named:

wheel

2. Enable GitHub Pages

Site will be:
https://yourusername.github.io/wheel/

3. Google Sheet Setup

Create sheet named:

activities

Columns:
emoji | activity | price | restaurant | movie | timestamp

4. Apps Script

Open Extensions → Apps Script

Paste Code.gs

Deploy → Web App

Access: Anyone

Copy Web App URL

5. Paste the Web App URL inside add.html where it says:

PASTE_YOUR_APPS_SCRIPT_URL_HERE

Now activities added from the form go into the sheet.
